# MPP Endpoint-as-a-Service

A service that enables acquirers to offer their merchants the ability to turn existing APIs into MPP (Machine Payments Protocol) card-compatible endpoints.

## What is this?

The Machine Payments Protocol (MPP) launched on March 18, 2026 as an open standard for machine-to-machine payments. Stripe merchants can accept MPP payments in a few lines of code. This project builds the equivalent capability for merchants on any acquirer.
